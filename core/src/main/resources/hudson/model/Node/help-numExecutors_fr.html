Le nombre maximum de build concurrent que Jenkins peut réaliser sur ce noeud.

Un nombre de builds simultanés supérieur à cette valeur augmentera la durée individuelle de chaque build. Néanmoins, la capacité globale peut être supérieure, puisqu'un processeur peut se charger d'un build pendant qu'un autre build est en attente sur les entrées/sorties.

Les agents (noeuds qui ne sont pas sur le contrôleur) doivent avoir au moins un exécuteur. Pour stopper temporairement les contructions sur un agent, utilisez le bouton Marquer ce noeud comme temporairement hors ligne sur la page de l'agent.

Pour le contrôleur (dont le noeud est appelé 'built-in'), une valeur de 0 garantit que le contrôleur ne déclenchera pas de build. Note: le contrôleur pourra toujours lancer des tâches légères ('flyweight task') y compris les tâches de haut niveau des pipelines.
